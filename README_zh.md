# HelloAdmin

HelloAdmin 是一个基于 Gin + Ant Design Vue UI 的前后端分离管理系统，前端使用 Ant Design Vue UI，后端使用 Gin 框架。

## 内置功能

- [x] JWT 登录
- [x] 角色管理
- [x] 菜单管理
- [x] 部门管理
- [x] 操作日志

体验地址: [http://demo.helloadmin.cn](http://demo.helloadmin.cn)
