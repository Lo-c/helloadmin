# HelloAdmin

基于 Gin + Ant Design Vue UI 前后端分离管理系统

## 功能模块

- [x] JWT 登录
- [x] 角色管理
- [x] 菜单管理
- [x] 部门管理
- [] Cabin 权限管理

体验地址: [http://demo.helloadmin.cn](http://demo.helloadmin.cn)
