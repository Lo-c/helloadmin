# helloadmin

基于Gin + Ant Design Vue UI 前后端分离管理系统

## 功能模块

- [x] JWT登录
- [x] 角色管理
- [x] 菜单管理
- [x] 管理员管理
- [x] 系统设置
- [x] 角色授权菜单

体验地址: [http://demo.helloadmin.cn](http://demo.helloadmin.cn)
